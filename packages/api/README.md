# Subsocial API by [DappForce](https://github.com/dappforce)

JavaScript API wrappers for Substrate and IPFS specific to Subsocial blockchain.

## Install

Using npm:

```bash
npm i @subsocial/api
```

Using yarn:

```bash
yarn add @subsocial/api
```
