# Subsocial Definitions by [DappForce](https://github.com/dappforce)

Subsocial definitions using @polkadot/typegen to generate type definitions

## Install

Using npm:

```bash
npm i @subsocial/definitions
```

Using yarn:

```bash
yarn add @subsocial/definitions
```
